# SuperClicker

SuperClicker è una semplice applicazione desktop sviluppata in **Java** con **Swing**.

L’applicazione mostra un pulsante e un contatore: ogni click incrementa il valore, che viene aggiornato in tempo reale nell’interfaccia grafica.

## Scopo
Progetto didattico pensato per:
- imparare le basi di Java Swing
- gestire eventi (ActionListener)
- comprendere la struttura di una GUI Java

## Tecnologie
- Java
- Swing

## Esecuzione
Apri il progetto con NetBeans (o un IDE compatibile), assicurati di avere il JDK installato ed esegui la classe `SuperClicker`.
